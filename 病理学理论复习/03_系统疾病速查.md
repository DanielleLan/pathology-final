# 系统疾病速查

> 每个疾病只抓四件事：**病因/机制 · 大体 · 镜下 · 临床病理联系**。病例题先看题眼，再回本文件定位。

---

## 一、心血管系统

### 动脉粥样硬化 AS

| 项目 | 高频点 |
|---|---|
| 定义 | 大、中动脉内膜脂质沉积、纤维组织增生，形成粥样斑块，使管壁增厚变硬、管腔狭窄 |
| 好发血管 | 主动脉、冠状动脉、脑动脉、肾动脉、四肢动脉 |
| 斑块成分 | 表面纤维帽；深部坏死崩解物、胆固醇结晶、泡沫细胞、钙化、炎细胞 |
| 继发改变 | 斑块破裂/溃疡、斑块内出血、血栓形成、钙化、动脉瘤形成 |
| 主动脉 | 腹主动脉最重，可形成动脉瘤并破裂 |
| 脑动脉 | 好发Willis环、大脑中动脉等，可脑萎缩、脑梗死、动脉瘤破裂出血 |
| 肾动脉 | 肾血管性高血压、肾梗死，瘢痕形成后动脉粥样硬化性固缩肾 |
| 四肢动脉 | 下肢多见，间歇性跛行，严重可足趾干性坏疽 |

### 冠心病与心肌梗死

| 考点 | 高频点 |
|---|---|
| 冠脉好发 | 左前降支最多，其次右冠、左主干/左旋支 |
| 冠心病类型 | 心绞痛、心肌梗死、心肌纤维化、冠状动脉性猝死 |
| 心绞痛 | 短暂心肌缺血缺氧，无心肌坏死；稳定型、非稳定型、变异型 |
| 心梗定义 | 冠脉持续闭塞导致心肌缺血性坏死 |
| 好发部位 | 左室前壁、心尖部、室间隔前2/3常见 |
| 大体演变 | 早期不明显；1-2天苍白/土黄；后充血出血带；数周后瘢痕 |
| 镜下 | 凝固性坏死，中性粒细胞浸润，后巨噬细胞清除，肉芽组织，瘢痕 |
| 并发症 | 心律失常、心衰、心源性休克、心破裂、室壁瘤、附壁血栓、心包炎 |

### 高血压病

| 项目 | 良性/缓进性高血压 |
|---|---|
| 基本病变 | 全身细小动脉硬化，尤其细动脉玻璃样变 |
| 心 | 左心室代偿性肥大，晚期扩张心衰 |
| 肾 | 入球小动脉玻璃样变，小动脉硬化；双肾细颗粒状，称原发性颗粒性固缩肾 |
| 脑 | 脑小动脉硬化，可发生脑出血；常见部位为基底节、内囊 |
| 视网膜 | 小动脉硬化、出血、渗出、视乳头水肿 |

| 项目 | 恶性/急进性高血压 |
|---|---|
| 人群 | 青壮年较多，舒张压显著升高 |
| 特征病变 | 坏死性细动脉炎、增生性小动脉硬化 |
| 主要累及 | 肾，易较早出现肾衰 |

### 风湿病

| 项目 | 高频点 |
|---|---|
| 本质 | 与A组乙型溶血性链球菌感染后免疫交叉反应有关的变态反应性疾病 |
| 基本病变 | 变质渗出期、增生期/肉芽肿期、瘢痕期 |
| 特征 | Aschoff小体：中心纤维素样坏死，周围Aschoff细胞/Anitschkow细胞 |
| 心内膜炎 | 二尖瓣最常受累，瓣膜闭锁缘形成小疣状白色血栓赘生物 |
| 心肌炎 | 心肌间质小血管旁Aschoff小体；可心率快、第一心音低钝 |
| 心外膜炎 | 浆液性或纤维素性心包炎，可绒毛心 |
| 关节 | 游走性多关节炎，常无后遗症 |
| 皮肤/脑 | 环形红斑、皮下结节；小舞蹈病 |

### 感染性心内膜炎 IE

| 项目 | 急性IE | 亚急性IE |
|---|---|---|
| 常见菌 | 金黄色葡萄球菌 | 草绿色链球菌 |
| 基础瓣膜 | 可发生在正常瓣膜 | 多在已有病变瓣膜 |
| 赘生物 | 大、松脆、破坏性强，易脱落 | 较小，菜花样，机化较明显 |
| 后果 | 瓣膜穿孔、腱索断裂、败血症、栓塞 | 瓣膜变形、栓塞、免疫复合物肾炎 |

### 慢性心瓣膜病

| 病变 | 常见原因 | 心脏形态/血流动力学 |
|---|---|---|
| 二尖瓣狭窄 | 风湿病 | 左房扩大，肺淤血，右心负荷增加；梨形心 |
| 二尖瓣关闭不全 | 风湿病、IE | 左房左室容量负荷增加；球形心 |
| 主动脉瓣狭窄 | 风湿病、钙化 | 左室压力负荷增加，向心性肥大 |
| 主动脉瓣关闭不全 | 风湿病、IE、梅毒性主动脉炎 | 左室容量负荷增加；可水冲脉、毛细血管搏动 |

## 二、呼吸系统

### 肺炎三类

| 项目 | 大叶性肺炎 | 小叶性肺炎 | 病毒性/间质性肺炎 |
|---|---|---|---|
| 性质 | 肺泡内纤维素性炎 | 急性化脓性炎 | 间质性炎 |
| 人群 | 青壮年 | 小儿、老人、久病卧床 | 儿童或成人均可 |
| 范围 | 一个肺叶/大部 | 细支气管为中心，多发小灶 | 肺泡间隔为主 |
| 典型阶段 | 充血水肿、红肝样、灰肝样、溶解消散 | 散在灰黄色实变灶，可融合 | 肺泡壁增宽，淋巴单核浸润 |
| 确诊题眼 | 红/灰肝样变 | 背侧下叶、小灶化脓 | 病毒包涵体 |
| 并发症 | 肺肉质变、胸膜炎、肺脓肿、败血症 | 肺脓肿、脓胸、支扩、心衰/呼衰 | 透明膜、继发细菌感染 |

### 慢性阻塞性肺疾病 COPD

| 疾病 | 高频点 |
|---|---|
| 慢性支气管炎 | 咳嗽咳痰每年≥3个月，连续2年以上；黏液腺增生，杯状细胞增多 |
| 肺气肿 | 终末细支气管远端气腔异常扩大，伴肺泡壁破坏；肺体积大，边缘钝圆，弹性差 |
| 支气管扩张 | 支气管持久扩张；反复咳大量脓痰、咯血，体位相关，杵状指 |
| 慢性肺心病 | 肺疾病导致肺动脉高压，右心室肥大扩张；右室壁厚度常>0.5cm |

### 尘肺

| 疾病 | 题眼 |
|---|---|
| 硅沉着病 | 游离二氧化硅；硅结节为同心圆/旋涡状玻璃样胶原纤维；肺纤维化；可并发肺结核 |
| 石棉沉着病 | 石棉小体；肺间质纤维化；与胸膜斑、间皮瘤、肺癌相关 |

### 鼻咽癌

| 项目 | 高频点 |
|---|---|
| 病因 | EBV关系密切，华南高发 |
| 部位 | 鼻咽顶侧壁、咽隐窝 |
| 组织学 | 非角化性癌常见；可分角化性鳞癌、非角化性癌等 |
| 转移 | 早期淋巴道转移至颈部淋巴结 |
| 题眼 | 鼻塞、涕血、耳鸣、颈部淋巴结肿大 |

### 肺癌

| 类型 | 高频特点 |
|---|---|
| 鳞状细胞癌 | 多中央型，与吸烟关系强；角化珠、细胞间桥；可空洞 |
| 腺癌 | 最常见；女性/不吸烟者相对多；周围型；可有贴壁样、腺管/乳头等结构 |
| 小细胞癌 | 神经内分泌肿瘤；中央型；燕麦样细胞；恶性高，早转移；可ACTH等副肿瘤综合征 |
| 大细胞癌 | 未分化非小细胞癌，排除性诊断 |
| 肺原位腺癌 | 纯贴壁样生长，无浸润，预后好 |

## 三、消化系统与肝胆胰

### 慢性胃炎

| 类型 | 高频点 |
|---|---|
| 慢性浅表性胃炎 | 胃窦多见，黏膜浅层炎细胞浸润 |
| 慢性萎缩性胃炎 | 腺体萎缩减少，肠上皮化生/假幽门腺化生，可有不典型增生；与胃癌相关 |
| A型 | 自身免疫，胃体胃底，壁细胞抗体，低酸/恶性贫血 |
| B型 | 幽门螺杆菌相关，我国多见，胃窦为主 |

### 消化性溃疡

| 项目 | 高频点 |
|---|---|
| 好发 | 十二指肠球部最多，其次胃小弯近幽门 |
| 大体 | 圆形/椭圆，边缘整齐，底部平坦，直径多≤2cm |
| 镜下四层 | 渗出层、坏死层、肉芽组织层、瘢痕层 |
| 并发症 | 出血、穿孔、幽门梗阻、癌变（胃溃疡可，十二指肠溃疡罕见） |
| 鉴别胃癌 | 良性溃疡边缘整齐、底平；癌性溃疡边缘隆起不整、底凹凸、周围黏膜皱襞中断 |

### 炎症性肠病

| 项目 | 溃疡性结肠炎 UC | Crohn病 |
|---|---|---|
| 部位 | 直肠开始，连续向近端 | 回肠末端常见，节段性跳跃 |
| 范围 | 黏膜和黏膜下层 | 全壁性炎 |
| 大体 | 浅表溃疡、假息肉 | 裂隙状溃疡、铺路石样、肠壁增厚狭窄 |
| 镜下 | 隐窝脓肿，腺体结构紊乱 | 非干酪样肉芽肿、全壁炎 |
| 并发 | 中毒性巨结肠、癌变 | 瘘管、狭窄、肠梗阻 |

### 消化道肿瘤

| 肿瘤 | 高频点 |
|---|---|
| 食管癌 | 鳞癌最常见；中段多；Barrett食管可进展为腺癌 |
| 胃癌 | 胃窦小弯侧多；早期胃癌限于黏膜/黏膜下；弥漫型可革囊胃；印戒细胞癌可Krukenberg转移 |
| 结直肠癌 | 直肠、乙状结肠多；腺瘤-癌序列；绒毛状腺瘤、FAP癌变风险高 |
| GIST | 胃肠道最常见间叶性肿瘤；常表达CD117、DOG1、CD34；危险度看大小、部位、核分裂像 |
| 肝细胞癌 | HBV/肝硬化背景；梁索状排列、血窦丰富；AFP常升高；肝内血道转移最常见 |
| 胆管癌 | 腺癌形态，需与转移性腺癌鉴别 |
| 胰腺癌 | 胰头多见，可无痛性黄疸；导管腺癌常见 |

### 病毒性肝炎

| 病变 | 高频点 |
|---|---|
| 点状坏死 | 单个或数个肝细胞坏死，急性普通型常见 |
| 碎片状坏死/界面性炎 | 炎细胞突破界板，慢性肝炎活动性改变 |
| 桥接坏死 | 坏死带连接汇管区/中央静脉，中重度慢性肝炎 |
| 大片坏死 | 急性/亚急性重型肝炎，可急性黄色肝萎缩 |
| 毛玻璃样肝细胞 | HBsAg蓄积，提示HBV感染 |

### 肝硬化

| 项目 | 高频点 |
|---|---|
| 定义 | 弥漫性肝细胞变性坏死、再生结节、纤维组织增生，形成假小叶 |
| 基本病变 | 假小叶形成：肝细胞再生结节 + 纤维隔 + 血管结构紊乱 |
| 临床联系 | 门脉高压、肝功能不全 |
| 门脉高压 | 脾大/脾亢、腹水、侧支循环开放、胃肠淤血 |
| 肝功能不全 | 黄疸、出血倾向、低蛋白水肿、肝性脑病、内分泌紊乱 |
| 血吸虫肝硬化 | 汇管区纤维化明显，干线型肝硬化，窦前性门脉高压，假小叶不明显 |

## 四、泌尿系统

### 肾小球疾病总表

| 类型 | 机制/题眼 | 临床 |
|---|---|---|
| 急性弥漫增生性GN | 感染后免疫复合物，毛细血管内皮/系膜增生，中性粒细胞 | 急性肾炎综合征 |
| 新月体性GN | 壁层上皮增生 + 单核细胞；>50%肾小球有新月体 | 急进性肾炎综合征 |
| 微小病变 | 光镜基本正常，电镜足突融合 | 儿童肾病综合征，激素敏感 |
| 膜性肾病 | 成人肾病综合征；基底膜弥漫增厚，钉突 | PLA2R抗体相关 |
| FSGS | 局灶节段硬化 | 肾病综合征，预后较差 |
| 膜增生性GN | 系膜增生，GBM双轨征 | 可肾炎或肾病综合征 |
| IgA肾病 | 系膜区IgA沉积 | 感染后短期肉眼血尿 |

### 肾盂肾炎

| 项目 | 急性肾盂肾炎 | 慢性肾盂肾炎 |
|---|---|---|
| 病因 | 多为大肠杆菌，上行感染常见 | 反复感染、尿路梗阻/反流 |
| 临床 | 发热寒战、腰痛、尿频尿急尿痛、脓尿 | 反复尿路感染、夜尿、高血压、肾衰 |
| 大体 | 肾表面散在黄白脓肿，肾盂黏膜充血脓性渗出 | 肾盏变形扩张，肾实质瘢痕，双肾不对称 |
| 镜下 | 中性粒细胞浸润，肾小管脓肿 | 间质纤维化，慢性炎细胞，肾小管甲状腺样变 |

### 肾肿瘤与尿路上皮肿瘤

| 肿瘤 | 高频点 |
|---|---|
| 肾透明细胞癌 | 成人最常见肾恶性肿瘤；来源近曲小管；金黄色、出血坏死；细胞透明富含脂质/糖原；可侵入肾静脉 |
| 乳头状肾细胞癌 | 乳头状结构，可与c-MET相关 |
| 肾嫌色细胞癌 | 预后较好 |
| 肾母细胞瘤 Wilms | 儿童常见；WT1阳性；小圆细胞、原始肾小球/肾小管样结构、间叶成分；切面灰白鱼肉状 |
| 尿路上皮癌 | 膀胱最常见恶性肿瘤；乳头状/菜花状，可血尿；分级看细胞异型和结构紊乱 |

## 五、生殖系统与乳腺

### 宫颈病变

| 项目 | 高频点 |
|---|---|
| CIN/SIL | 高危HPV持续感染，鳞柱交界区；绝经后病变常向颈管内移 |
| 宫颈癌 | 鳞癌最常见，其次腺癌；HPV 16/18等相关；可外生菜花状、内生浸润、溃疡型 |
| 转移 | 直接蔓延、淋巴道转移，晚期可血道 |
| HPV 6/11 | 尖锐湿疣为主，一般低危 |

### 子宫内膜和滋养层疾病

| 疾病 | 高频点 |
|---|---|
| 子宫内膜增生/癌 | 与长期雌激素刺激相关；绝经后阴道流血是题眼；内膜样腺癌常见 |
| 子宫平滑肌瘤 | 育龄女性，雌激素相关；境界清楚、编织状平滑肌束，异型少 |
| 葡萄胎 | 绒毛水肿、滋养层增生，hCG升高，葡萄串样 |
| 侵袭性葡萄胎 | 有绒毛结构，侵入肌层/转移 |
| 绒毛膜癌 | 无绒毛结构，出血坏死明显，血道转移早，hCG显著升高 |

### 乳腺

| 病变 | 高频点 |
|---|---|
| 纤维囊性乳腺病 | 多灶小结节、界限不清、疼痛与月经周期相关；囊肿、纤维化、腺病等 |
| 纤维腺瘤 | 年轻女性，单发活动结节；上皮和纤维间质双成分增生，边界清楚 |
| 导管原位癌 DCIS | 导管内癌细胞增生，未突破基底膜；粉刺癌有中央坏死 |
| 浸润性导管癌 | 女性最常见乳腺恶性肿瘤；硬、不活动、边界不清，可橘皮样改变 |
| 浸润性小叶癌 | 可单列索状浸润，常多灶/双侧 |
| 免疫标志 | ER/PR阳性提示内分泌治疗；HER2阳性可考虑曲妥珠单抗；Ki-67看增殖 |

### 卵巢常见题眼

| 肿瘤 | 高频点 |
|---|---|
| 成熟囊性畸胎瘤 | 多种成熟组织，如皮肤、神经、软骨、骨；可见钙化 |
| 未成熟畸胎瘤 | 含未成熟神经外胚叶等成分，恶性潜能 |
| 卵黄囊瘤 | Schiller-Duval小体，AFP升高 |
| Krukenberg瘤 | 胃肠道黏液癌/印戒细胞癌转移至卵巢，常双侧 |

## 六、淋巴造血、内分泌、神经、骨关节

### 淋巴造血系统

| 疾病 | 高频点 |
|---|---|
| 霍奇金淋巴瘤 | R-S细胞/镜影细胞；经典型四型：结节硬化、混合细胞、富于淋巴细胞、淋巴细胞消减 |
| 弥漫大B细胞淋巴瘤 | 大细胞弥漫浸润，侵袭性，CD20阳性 |
| 滤泡性淋巴瘤 | B细胞，滤泡结构，常BCL2相关 |
| Burkitt淋巴瘤 | 高度恶性B细胞，星空现象，MYC，常EBV相关 |
| CLL/SLL | 同一疾病不同表现：外周血为主叫CLL，组织肿块为主叫SLL |
| CML | BCR-ABL/费城染色体，粒系增生 |
| 髓样肉瘤 | 髓系肿瘤在骨髓外形成肿块 |

### 内分泌

| 疾病 | 高频点 |
|---|---|
| 结节性甲状腺肿 | 多发结节，滤泡大小不一，胶质多少不等，出血/囊变/钙化；无完整包膜 |

| 糖尿病 | 1型自身免疫破坏β细胞；2型胰岛素抵抗；并发大血管AS、小血管玻璃样变、肾病、视网膜病、神经病 |

## 七、传染病与寄生虫病

### 结核病

| 项目 | 高频点 |
|---|---|
| 基本病变 | 渗出、增生、干酪样坏死 |
| 结核结节 | 上皮样细胞、Langhans巨细胞、淋巴细胞，中央可干酪样坏死 |
| 原发综合征 | 肺原发灶 + 淋巴管炎 + 肺门淋巴结结核 |
| 继发性肺结核类型 | 局灶型、浸润型、慢性纤维空洞型、干酪样肺炎、结核球、结核性胸膜炎 |
| 开放性肺结核 | 慢性纤维空洞型等可经支气管排菌 |
| 结核球鉴别 | 周围型肺癌、炎性假瘤、真菌球、良性肿瘤等 |
| 肠结核 | 回盲部，环形溃疡，长轴与肠管垂直 |
| 肾结核 | 多血源播散，常单侧，皮髓交界开始，膀胱刺激征、脓尿、血尿 |

### 伤寒、菌痢、梅毒、麻风、真菌

| 疾病 | 高频点 |
|---|---|
| 伤寒 | 回肠末端集合淋巴小结；伤寒肉芽肿；溃疡长轴与肠管平行；可肠出血/穿孔 |
| 细菌性痢疾 | 乙状结肠/直肠，假膜性炎，黏液脓血便；中毒性菌痢肠病变轻全身中毒重 |
| 梅毒 | 一期硬下疳；二期梅毒疹；三期树胶肿；镜下浆细胞恒定出现，闭塞性动脉内膜炎 |
| 麻风 | 结核样型免疫强、肉芽肿、菌少；瘤型免疫弱、泡沫细胞多、菌多 |
| 隐球菌 | 易累及CNS，脑膜炎，可与结核性脑膜炎混淆 |
| 曲霉 | 肺常见，可见分隔菌丝锐角分支 |
| 毛霉 | 鼻脑型、血管侵袭，宽大无隔菌丝 |

### 寄生虫

| 疾病 | 高频点 |
|---|---|
| 阿米巴肠病 | 盲肠/升结肠，烧瓶状溃疡，液化性坏死，可见滋养体 |
| 阿米巴肝脓肿 | 多右叶，巧克力酱/果酱样坏死物，壁薄，脓液中可检滋养体 |
| 阿米巴痢疾 vs 菌痢 | 阿米巴口小底大烧瓶状、组织溶解；菌痢假膜性炎、表浅不规则溃疡 |
| 血吸虫病 | 虫卵引起肉芽肿和纤维化；急性虫卵结节有嗜酸性脓肿/Hoeppli现象，慢性为纤维化虫卵结节 |
| 干线型肝硬化 | 血吸虫虫卵沉积汇管区导致粗大纤维化，窦前性门脉高压 |
| 华支睾吸虫 | 胆管上皮增生、胆管炎、胆管周围纤维化，可与胆管癌相关 |
