## 两天复习路线

### Day 1：总论 + 高频系统

- [ ] 先读 [[04_题型病例真题#答题模板]]，知道病例题怎么拆。
- [ ] 背 [[02_机制对比名解#名词解释高频库]] 的 A 级名解。
- [ ] 背 [[01_高频病变核心速查#一、总论必考病变]]：适应/损伤/修复、血栓栓塞梗死、炎症、肿瘤。
- [ ] 背 [[03_系统疾病速查#一、心血管系统]]：动脉粥样硬化、高血压、风湿病、感染性心内膜炎、心梗。
- [ ] 背 [[03_系统疾病速查#二、呼吸系统]]：肺炎三类、肺气肿/肺心病、肺癌、鼻咽癌。
- [ ] 背 [[03_系统疾病速查#三、消化系统与肝胆胰]]：消化性溃疡、肝炎、肝硬化、胃癌/肠癌/肝癌。
- [ ] 用 [[10_闪卡]] 做一轮主动回忆。

### Day 2：系统补齐 + 真题病例

- [ ] 背 [[03_系统疾病速查#四、泌尿系统]]：肾炎综合征、新月体、肾盂肾炎、肾肿瘤。
- [ ] 背 [[03_系统疾病速查#五、生殖系统与乳腺]]：CIN/宫颈癌、滋养细胞疾病、乳腺癌。
- [ ] 背 [[03_系统疾病速查#六、淋巴造血、内分泌、神经、骨关节]]：霍奇金、CML、甲状腺、脑炎/胶质瘤、骨肉瘤。
- [ ] 背 [[03_系统疾病速查#七、传染病与寄生虫病]]：结核、伤寒、菌痢、梅毒、阿米巴、血吸虫。
- [ ] 过 [[04_题型病例真题#病例反查总表]]，每个病例只练“诊断 + 依据 + 机制/并发症”。
- [ ] 再背 [[02_机制对比名解#简答题专项]]，尤其是肿瘤、坏疽、癌肉瘤、血栓栓塞、肝硬化、肺炎、结核。
- [ ] 最后用 [[10_闪卡]] 快速查漏。

## 高频优先级

| 优先级 | 模块 | 理由 |
|---|---|---|
| A 必背 | 细胞损伤、血栓/栓塞/梗死、炎症、肿瘤总论 | 往年选择、名解、简答反复出现；也是病例题通用语言 |
| A 必背 | 心血管、呼吸、消化/肝、传染病 | 大纲学时高，病例和简答题密集 |
| A 必背 | 泌尿、生殖乳腺、淋巴造血 | 往年题出现新月体肾炎、乳腺癌、CIN、霍奇金、CML |
| B 要会 | 内分泌、神经、骨关节、寄生虫 | 选择和名解常考，简答压缩背 |
| C 了解 | 罕见肿瘤亚型、非常细的分子分型 | 只记典型关键词，除非老师课堂强调 |

## 考前最后 6 小时

1. 先背名解 A 级：坏死、坏疽、血栓形成、栓塞、梗死、肉芽组织、肉芽肿、原位癌、癌前病变、异型性、风湿小体、假小叶、新月体、R-S 细胞。
2. 再背 10 个简答：癌 vs 肉瘤、良恶性肿瘤、肿瘤扩散、坏疽类型、血栓类型、肺动脉栓塞 vs 体循环栓塞、肺炎三类、肝硬化三类、结核基本病变、蜂窝织炎 vs 脓肿。
3. 最后看病例反查：高血压脑出血、肺癌、乳腺癌、肝癌合并肝硬化、新月体肾炎、宫颈癌/滋养细胞疾病、慢性肺心病、结核、肾盂肾炎。

## 最省时间的闭环

```text
题型模板 → 高频总论 → 系统病变 → 名解/简答 → 病例反查 → 错点回补教材
```

不要把“看懂”当成“会考”。病理理论卷最吃香的是：**看到关键词能下诊断，问形态能写三条，问鉴别能列成表。**
