# 高频病变核心速查

> 用法：选择题看“题眼”；名解和简答看“最短可写答案”；病例题按“病变 → 形态 → 临床联系”反推。

---

## 一、总论必考病变

### 1. 适应：萎缩、肥大、增生、化生

| 病变 | 定义关键词 | 题眼 | 易错点 |
|---|---|---|---|
| 萎缩 | 发育正常的细胞/组织/器官体积缩小，可伴细胞数减少 | 脂褐素增多、器官体积小、色深、质韧 | 青春期后胸腺萎缩是生理性；压迫/失用/去神经/营养不良是病理性 |
| 肥大 | 细胞体积增大导致器官体积增大 | 高血压心肌肥大 | 哺乳期乳腺、妊娠子宫、运动员肌肉多为生理性 |
| 增生 | 实质细胞数目增多 | 前列腺增生、再生性增生、缺碘甲状腺增生 | 增生可为生理或病理，不等于肿瘤 |
| 化生 | 一种成熟细胞被另一种成熟细胞取代 | 支气管鳞化、胃黏膜肠上皮化生、宫颈鳞化 | 只发生在同源细胞之间；成纤维细胞变纤维细胞不是化生 |

**常考一句话**：化生增强局部耐受刺激的能力，但削弱原有功能，长期可癌变。

### 2. 可逆性损伤：变性和蓄积

| 病变 | 本质 | 好发/题眼 |
|---|---|---|
| 细胞水肿 | ATP下降，Na+-K+泵障碍，水钠入胞 | 心、肝、肾实质细胞；胞质淡染清亮 |
| 脂肪变 | 甘油三酯蓄积于非脂肪细胞 | 肝最常见；心肌脂变可见虎斑心；苏丹染色可证实 |
| 玻璃样变 | 均质红染、毛玻璃样蛋白蓄积 | 肾小管上皮重吸收蛋白过多；浆细胞 Russell 小体；高血压细动脉壁玻璃样变 |
| 淀粉样变 | 细胞外蛋白-黏多糖复合物沉积 | HE淡红，刚果红阳性；间质内沉积 |
| 黏液样变 | 间质内黏多糖/蛋白质蓄积 | 间质疏松、灰蓝色黏液样基质 |
| 脂褐素 | 自噬溶酶体内未消化细胞器残体 | 衰老/萎缩细胞，黄褐色颗粒 |
| 含铁血黄素 | 血红蛋白分解产物 | 陈旧性出血，普鲁士蓝阳性 |
| 营养不良性钙化 | 坏死/变性组织内钙盐沉积，血钙正常 | 干酪样坏死、动脉粥样硬化斑块 |
| 转移性钙化 | 钙磷代谢障碍，高血钙/高钙磷乘积 | 正常组织多发钙化 |

### 3. 坏死、凋亡、坏疽

| 项目 | 必背点 |
|---|---|
| 坏死定义 | 活体内局部细胞死亡，以酶溶性变化为特点，可引起炎症 |
| 光镜判断坏死 | **细胞核改变最重要**：核固缩、核碎裂、核溶解 |
| 凝固性坏死 | 心、肾、脾梗死常见；组织轮廓可保留 |
| 液化性坏死 | 脑、脓肿、胰腺脂肪坏死；组织液化成腔 |
| 干酪样坏死 | 结核特征；肉眼奶酪/豆渣样，镜下结构全消失 |
| 脂肪坏死 | 急性胰腺炎酶解性；皮下外伤性 |
| 纤维素样坏死 | 风湿病、结缔组织病、急进性高血压；血管壁/结缔组织红染 |
| 坏疽 | 较大范围坏死 + 腐败菌感染；干性、湿性、气性 |
| 凋亡 | 单个/小团细胞程序性死亡，膜不破裂，无急性炎症 |

**坏疽三型速记**

| 类型 | 好发 | 条件 | 形态 | 全身中毒 |
|---|---|---|---|---|
| 干性坏疽 | 四肢末端 | 动脉阻塞、静脉通畅 | 干缩、黑褐、分界清 | 轻 |
| 湿性坏疽 | 肠、肺、子宫、胆囊 | 动脉和静脉均受阻 | 湿润肿胀、恶臭、分界不清 | 重 |
| 气性坏疽 | 深部肌肉创伤 | 产气荚膜梭菌等厌氧菌 | 蜂窝状、捻发音 | 很重 |

### 4. 修复：肉芽组织和创伤愈合

| 考点 | 最短答案 |
|---|---|
| 再生 | 损伤周围同种细胞增生修复缺损 |
| 不稳定细胞 | 表皮、呼吸道/消化道黏膜上皮、淋巴造血细胞，强再生 |
| 稳定细胞 | 肝、肾小管、腺体、间叶细胞，平时静止，损伤后可增生 |
| 永久细胞 | 神经元、心肌、骨骼肌，再生能力很弱 |
| 肉芽组织 | 新生薄壁毛细血管 + 成纤维细胞 + 炎细胞；鲜红颗粒状、柔软湿润 |
| 肉芽组织作用 | 抗感染保护创面；填补缺损；机化/包裹坏死组织、血栓、异物 |
| 一期愈合 | 缺损小、创缘整齐、无感染、对合好；瘢痕少 |
| 二期愈合 | 缺损大、创缘不整/感染；肉芽组织多、时间长、瘢痕大 |

### 5. 局部血液循环障碍

| 病变 | 定义/题眼 | 高频点 |
|---|---|---|
| 充血 | 动脉输入血量增多，主动过程 | 局部红、热；炎症性充血、减压后充血 |
| 淤血 | 静脉回流受阻，被动过程 | 暗红/发绀、水肿、出血、实质萎缩、间质纤维化 |
| 慢性肺淤血 | 多由左心衰引起 | 肺泡壁毛细血管扩张，心衰细胞，肺褐色硬化 |
| 慢性肝淤血 | 多由右心衰引起 | 小叶中央淤血，周边脂肪变，槟榔肝 |
| 血栓形成 | 活体心血管内血液凝固或有形成分凝集成固体质块 | 条件：内皮损伤、血流改变、血液高凝 |
| 栓塞 | 不溶性异常物随血流运行阻塞血管 | 血栓栓塞最常见；羊水栓塞题眼是分娩中突发呼吸困难休克 |
| 梗死 | 血管阻塞、血流停止导致局部缺血性坏死 | 形状取决于血管分布；类型看器官结构和淤血状态 |

**血栓类型**

| 类型 | 好发 | 成分/形态 |
|---|---|---|
| 白色血栓 | 心瓣膜、心腔、动脉，血流快处 | 血小板和少量纤维蛋白，灰白、坚实、黏附 |
| 混合血栓 | 静脉延续性血栓体部、动脉瘤、附壁血栓 | 红白相间层状；血小板梁 + 红细胞 |
| 红色血栓 | 静脉血栓尾部 | 红细胞多，湿润，易脱落 |
| 透明血栓 | 微循环小血管，DIC | 纤维蛋白为主，显微镜下见 |

**栓塞方向**

| 来源 | 去向 |
|---|---|
| 下肢深静脉/右心血栓 | 肺动脉栓塞 |
| 左心/主动脉系统血栓 | 脑、肾、脾、下肢等体循环动脉栓塞 |
| 门静脉系统血栓 | 肝内门静脉分支 |

**梗死类型**

| 类型 | 好发 | 条件/形态 |
|---|---|---|
| 贫血性梗死 | 心、肾、脾 | 实质致密、侧支少，凝固性坏死，灰白楔形 |
| 出血性梗死 | 肺、肠 | 双重血供/侧支丰富、组织疏松、严重淤血，暗红 |
| 败血性梗死 | 含菌栓子阻塞 | 梗死灶伴脓肿形成 |

### 6. 炎症

| 考点 | 必背点 |
|---|---|
| 炎症定义 | 具有血管系统的活体组织对损伤因子的防御反应，血管反应是中心环节 |
| 基本病变 | 变质、渗出、增生 |
| 急性炎症 | 中性粒细胞为主；血管通透性增加、渗出明显 |
| 慢性炎症 | 淋巴细胞/浆细胞/巨噬细胞浸润，组织破坏，修复纤维化 |
| 病毒感染 | 多为淋巴细胞、单核细胞浸润 |
| 化脓菌感染 | 中性粒细胞浸润 |
| 发热介质 | IL-1、TNF、前列腺素等 |

**炎症形态类型**

| 类型 | 关键词 | 例子 |
|---|---|---|
| 变质性炎 | 变性坏死突出 | 病毒性肝炎、乙脑 |
| 浆液性炎 | 稀薄浆液渗出 | 烫伤水疱、感冒初期黏膜 |
| 纤维素性炎 | 纤维素渗出 | 白喉、细菌性痢疾、绒毛心 |
| 化脓性炎 | 中性粒细胞 + 脓液 | 表面化脓、积脓、蜂窝织炎、脓肿 |
| 出血性炎 | 渗出物含大量红细胞 | 鼠疫、炭疽、流行性出血热 |
| 肉芽肿性炎 | 巨噬细胞及其衍生细胞结节状增生 | 结核、麻风、梅毒、异物、结节病、真菌/寄生虫 |

### 7. 肿瘤总论

| 考点 | 高频答案 |
|---|---|
| 肿瘤 | 局部组织细胞在基因改变基础上形成的异常克隆性增生，去除刺激后仍持续生长 |
| 异型性 | 肿瘤组织结构和细胞形态与相应正常组织的差异 |
| 分化 | 肿瘤细胞接近其来源正常细胞的程度；分化越低，异型性越大，恶性程度越高 |
| 间变 | 恶性肿瘤细胞缺乏分化，异型性显著 |
| 原位癌 | 上皮全层癌变，但未突破基底膜 |
| 浸润癌 | 癌细胞突破基底膜向间质浸润 |
| 癌前病变 | 具有癌变潜能的病变，如黏膜白斑、CIN、慢性萎缩性胃炎、Barrett食管、UC、结直肠腺瘤/FAP |
| 肿瘤分级 | 看分化程度、异型性、核分裂像 |
| 肿瘤分期 | 看肿瘤大小/浸润深度、淋巴结、远处转移 |
| 癌标记 | CK/CK7/CK20常用于上皮来源；LCA/CD45用于淋巴造血；CD3 T细胞；CD20 B细胞 |

**良恶性鉴别**

| 项目 | 良性 | 恶性 |
|---|---|---|
| 分化/异型性 | 分化好、异型性小 | 分化差、异型性大 |
| 生长速度 | 慢 | 较快，可坏死出血 |
| 生长方式 | 膨胀性/外生性，有包膜 | 浸润性，边界不清 |
| 转移 | 不转移 | 可转移 |
| 复发 | 少 | 多 |
| 对机体影响 | 局部压迫/激素 | 破坏、出血坏死、恶病质、副肿瘤综合征 |

**癌 vs 肉瘤**

| 项目 | 癌 | 肉瘤 |
|---|---|---|
| 来源 | 上皮组织 | 间叶组织 |
| 发病 | 常见，多见中老年 | 较少，多见青少年/儿童相对多 |
| 大体 | 灰白、质硬 | 灰红、鱼肉状、质软 |
| 镜下 | 癌巢，间质分界较清 | 弥漫分布，间质血管丰富 |
| 转移 | 淋巴道常见 | 血道常见 |
| 免疫 | CK常阳性 | Vimentin常阳性 |

## 二、系统病变一页总览

| 系统 | 必背病变 | 题眼 |
|---|---|---|
| 心血管 | 动脉粥样硬化、冠心病/心梗、高血压、风湿病、感染性心内膜炎、瓣膜病 | 粥样斑块继发改变、Aschoff小体、瓣膜赘生物、颗粒性固缩肾、内囊出血 |
| 呼吸 | 大叶/小叶/病毒性肺炎、肺气肿、支扩、肺心病、硅肺、鼻咽癌、肺癌 | 肺肉质变、包涵体、硅结节、EBV、肺小细胞癌神经内分泌 |
| 消化/肝 | 慢性萎缩性胃炎、消化性溃疡、胃癌/肠癌、病毒性肝炎、肝硬化、肝癌 | 溃疡四层、假小叶、桥接坏死、毛玻璃样肝细胞、AFP |
| 泌尿 | 肾炎综合征、新月体肾炎、肾病综合征、肾盂肾炎、肾细胞癌、Wilms瘤、尿路上皮癌 | 抗GBM、PLA2R、新月体>50%、透明细胞癌、WT1 |
| 生殖乳腺 | CIN/宫颈癌、葡萄胎/绒癌、乳腺癌、子宫内膜癌、卵巢肿瘤 | HPV、hCG、无绒毛结构、ER/PR、HER2 |
| 淋巴造血 | 霍奇金淋巴瘤、非霍奇金淋巴瘤、CML | R-S细胞、镜影细胞、星空现象、BCR-ABL |
| 内分泌 | 甲状腺肿、Graves、甲状腺乳头状癌/髓样癌、糖尿病 | 乳头+毛玻璃核+砂粒体，髓样癌=神经内分泌/C细胞 |
| 神经 | 化脓性脑膜炎、乙脑、胶质瘤、AD、PD | 蛛网膜下腔脓性渗出、噬神经细胞、Lewy小体 |
| 骨关节 | 骨肉瘤、骨巨细胞瘤、软骨肉瘤、骨质疏松 | 肿瘤性骨样组织、Codman三角、骨端 |
| 传染/寄生虫 | 结核、伤寒、菌痢、梅毒、阿米巴、血吸虫 | 干酪样坏死、肠溃疡方向、浆细胞、烧瓶状溃疡、干线型肝硬化 |
