# 机制 · 对比 · 名词解释

> 主观题用这个文件。名词解释按“三句式”背：**定义一句 + 形态/机制一句 + 例子/意义一句**。简答题按“先总后分、先定义后形态、最后临床联系”写。

## 名词解释高频库

### A 级：必须会写

| 名词 | 三句式答案 |
|---|---|
| 适应 adaptation | 细胞、组织或器官对持续刺激作出的非损伤性应答。形态上表现为萎缩、肥大、增生、化生。可为生理性或病理性。 |
| 萎缩 atrophy | 发育正常的细胞、组织或器官体积缩小，可伴实质细胞数减少。镜下细胞变小、胞质脂褐素增多，间质可增生。常见于营养不良、压迫、失用、去神经、内分泌不足。 |
| 化生 metaplasia | 一种分化成熟的细胞被另一种分化成熟的同源细胞取代。常见如支气管鳞状上皮化生、胃黏膜肠上皮化生。长期刺激下可增加癌变风险。 |
| 变性 degeneration | 细胞或间质受损后因代谢障碍发生异常物质或正常物质异常蓄积。属于可逆性损伤。包括水肿、脂肪变、玻璃样变、淀粉样变、黏液样变等。 |
| 坏死 necrosis | 活体内局部细胞死亡，以酶溶性变化为特点。镜下核固缩、核碎裂、核溶解是主要标志。常引起炎症反应。 |
| 凋亡 apoptosis | 活体内单个或小团细胞由基因调控的程序性死亡。细胞膜完整，形成凋亡小体，被吞噬细胞清除。一般不引起急性炎症。 |
| 坏疽 gangrene | 较大范围组织坏死并继发腐败菌感染。分为干性、湿性和气性坏疽。常见于四肢、肠、肺、子宫、深部创伤肌肉等。 |
| 机化 organization | 新生肉芽组织长入并取代坏死组织、血栓、脓液或异物的过程。最终常形成纤维瘢痕。是坏死、血栓等结局之一。 |
| 肉芽组织 granulation tissue | 由新生薄壁毛细血管、增生的成纤维细胞和炎细胞组成。肉眼鲜红、颗粒状、柔软湿润。作用为抗感染、填补缺损、机化或包裹异物。 |
| 血栓形成 thrombosis | 活体心血管内血液凝固或有形成分凝集形成固体质块的过程。基本条件是内皮损伤、血流状态改变、血液高凝。血栓可溶解吸收、机化、再通、钙化或脱落栓塞。 |
| 血栓 thrombus | 活体心脏或血管内形成的固体质块。按成分和部位分为白色、混合、红色和透明血栓。与死后血凝块不同，血栓常与血管壁黏连。 |
| 栓塞 embolism | 循环血液中不溶性异常物质随血流运行并阻塞血管的过程。阻塞物称栓子。血栓栓塞最常见，还可见脂肪、气体、羊水、肿瘤细胞栓塞。 |
| 梗死 infarction | 器官或局部组织因血管阻塞、血流停止导致缺血缺氧而发生的坏死。分贫血性和出血性梗死。心、肾、脾多为贫血性，肺、肠常为出血性。 |
| 炎症 inflammation | 具有血管系统的活体组织对损伤因子的防御反应。基本病变为变质、渗出、增生。血管反应是炎症中心环节。 |
| 渗出 exudation | 炎症时血管内液体、蛋白质和炎细胞通过血管壁进入组织间隙、体腔或黏膜表面的过程。是急性炎症的重要表现。与血管通透性升高和白细胞游出有关。 |
| 蜂窝织炎 phlegmon/cellulitis | 疏松结缔组织的弥漫性化脓性炎。常由溶血性链球菌引起，病变弥漫、边界不清。常见于皮下、肌间、阑尾等。 |
| 脓肿 abscess | 器官或组织内局限性化脓性炎。局部组织坏死溶解形成脓腔，腔内充满脓液。常由金黄色葡萄球菌引起。 |
| 肉芽肿 granuloma | 局部巨噬细胞及其衍生细胞增生形成境界较清的结节状病灶。可含上皮样细胞、多核巨细胞、淋巴细胞和纤维组织。常见于结核、麻风、梅毒、异物、真菌等。 |
| 菌血症 bacteremia | 细菌由局部病灶进入血流，但不在血中大量繁殖，临床可无明显中毒症状。血培养可检出细菌。常为短暂性。 |
| 毒血症 toxemia | 细菌毒素或毒性产物吸收入血引起全身中毒症状。血中通常无大量细菌。可表现为高热、寒战等。 |
| 败血症 septicemia | 细菌入血后大量繁殖并产生毒素，引起全身中毒症状和病理变化。可有高热、皮肤黏膜出血点、脾和淋巴结肿大。血培养常阳性。 |
| 脓毒败血症 pyemia | 化脓菌引起败血症，同时在全身多器官形成多发性栓塞性脓肿。常与含菌血栓栓子有关。 |
| 异型性 atypia | 肿瘤组织结构和细胞形态与相应正常组织的差异。异型性越大，分化越差，恶性程度通常越高。包括结构异型性和细胞异型性。 |
| 间变 anaplasia | 恶性肿瘤细胞高度缺乏分化的状态。表现为细胞和核明显异型、病理性核分裂像多、极性消失。提示高度恶性。 |
| 原位癌 carcinoma in situ | 上皮全层发生癌变，但癌细胞尚未突破基底膜。无间质浸润和转移。突破基底膜后称浸润癌。 |
| 癌前病变 precancerous lesions | 具有较高癌变风险的病变。常见如黏膜白斑、CIN、Barrett食管、慢性萎缩性胃炎、慢性溃疡性结肠炎、结直肠腺瘤和家族性腺瘤性息肉病。及时处理可阻断癌变。 |
| 转移 metastasis | 恶性肿瘤细胞从原发部位侵入淋巴管、血管或体腔，到达远处继续生长形成同类型肿瘤。途径包括淋巴道、血道和种植性转移。转移是恶性肿瘤重要标志。 |
| 风湿小体 Aschoff body | 风湿病特征性肉芽肿样病变。中心为纤维素样坏死，周围有Aschoff细胞、Anitschkow细胞及炎细胞。多见于心肌间质。 |
| 心衰细胞 heart failure cell | 慢性肺淤血时，肺泡腔内吞噬含铁血黄素的巨噬细胞。提示红细胞漏出并被巨噬细胞吞噬分解。常见于左心衰。 |
| 肺肉质变 pulmonary carnification | 大叶性肺炎时肺泡内纤维素渗出物未被溶解吸收，而被肉芽组织机化。病变肺呈褐色肉样。属于大叶性肺炎并发症。 |
| 桥接坏死 bridging necrosis | 肝内坏死带连接两个汇管区、两个中央静脉或汇管区与中央静脉。常见于中、重度慢性肝炎。提示病变较重。 |
| 碎片状坏死/界面性炎 piecemeal necrosis/interface hepatitis | 汇管区炎细胞突破界板，向肝小叶内侵袭并导致界板肝细胞坏死。常见于慢性肝炎。是慢性肝炎活动的重要形态。 |
| 毛玻璃样肝细胞 ground-glass hepatocyte | HBV感染时肝细胞胞质内HBsAg蓄积，HE呈均质淡染毛玻璃样。提示乙肝病毒感染。 |
| 假小叶 pseudolobule | 肝硬化时增生纤维组织分割原肝小叶并包绕再生肝细胞团形成的结构。小叶结构和血管关系紊乱。是假小叶形成的病理基础。 |
| R-S细胞 Reed-Sternberg cell | 霍奇金淋巴瘤的诊断性肿瘤细胞。体积大，双核或多核，核仁大而嗜酸，双核对称时称镜影细胞。背景常有多种炎细胞。 |
| 新月体 crescent | 肾小囊壁层上皮细胞增生并伴单核细胞渗出，在毛细血管球外形成新月形结构。超过50%肾小球受累可诊断新月体性肾炎。早期为细胞性，晚期可纤维化。 |
| 肾病综合征 nephrotic syndrome | 大量蛋白尿、低白蛋白血症、明显水肿、高脂血症/脂尿。常见病理类型有微小病变、膜性肾病、FSGS、膜增生性肾炎等。 |
| CIN | 子宫颈上皮内瘤变，是宫颈鳞状上皮癌前病变。多发生于鳞柱交界区，与高危HPV感染有关。可分低级别和高级别鳞状上皮内病变。 |
| 侵袭性葡萄胎 invasive mole | 葡萄胎组织侵入子宫肌层或转移到远处。镜下仍可见水肿绒毛和滋养层细胞增生。血hCG升高。 |
| 绒毛膜癌 choriocarcinoma | 高度恶性的滋养层细胞肿瘤。由肿瘤性细胞滋养层和合体滋养层组成，**无绒毛结构**，常出血坏死明显。血hCG显著升高，早期血道转移。 |

### B 级：常考但可压缩

| 名词 | 三句式答案 |
|---|---|
| 白色血栓 | 血流较快处形成，以血小板和少量纤维蛋白为主。常位于心瓣膜、心腔和动脉，是延续性血栓头部。肉眼灰白、质实、与壁黏连。 |
| 混合血栓 | 由血小板梁、纤维蛋白、红细胞等混合构成。常见于静脉延续性血栓体部、动脉瘤和附壁血栓。肉眼呈红白相间层状。 |
| 透明血栓 | 发生于微循环小血管内，以纤维蛋白为主。常见于DIC。只能在显微镜下观察到。 |
| 绒毛心 | 纤维素性心包炎时，心包膜表面纤维素渗出呈绒毛状。常见于风湿病、尿毒症等。属于纤维素性炎。 |
| 硅结节 | 矽肺的特征性病变。由同心圆或旋涡状排列的玻璃样变胶原纤维构成，中央可有坏死。是长期吸入游离二氧化硅所致。 |
| 小细胞肺癌 | 肺神经内分泌肿瘤，恶性程度高。镜下细胞小、胞质少、核深染，可呈燕麦样。常中央型，早期转移，常伴副肿瘤综合征。 |
| 肺原位腺癌 | 局限性、纯贴壁样生长的肺腺癌前/早期病变，未见间质、血管或胸膜浸润。预后好。常为周围型小结节或磨玻璃影。 |
| 粉刺癌 comedocarcinoma | 导管原位癌的一型，导管内癌细胞增生伴中央坏死，切面可挤出粉刺样坏死物。可伴钙化。提示乳腺导管原位癌。 |
| 小叶原位癌 LCIS | 乳腺小叶终末导管-小叶单位内小而一致的肿瘤细胞增生，未突破基底膜。常为多灶、双侧。是乳腺癌风险标志。 |
| 革囊胃 linitis plastica | 弥漫浸润型胃癌使胃壁弥漫增厚变硬、胃腔缩小，外观似皮革袋。常与印戒细胞癌相关。预后差。 |
| 早期胃癌 | 癌组织局限于黏膜层或黏膜下层，无论有无淋巴结转移。与肿块大小无直接等同关系。预后较好。 |
| 细胞性新月体 | 新月体早期阶段，主要由壁层上皮细胞增生和单核细胞渗出构成。提示急进性肾小球肾炎活动期。可发展为纤维性新月体。 |
| BCR-ABL融合基因 | 9号和22号染色体易位形成的融合基因，产生异常酪氨酸激酶。典型见于慢性髓细胞性白血病。对应费城染色体。 |
| 星空现象 starry-sky | Burkitt淋巴瘤中肿瘤细胞高度增殖，散在吞噬凋亡细胞的巨噬细胞呈淡染“星星”。背景深染肿瘤细胞形成“夜空”。 |
| 垂体性巨人症 | 骺板闭合前生长激素过多导致全身骨骼和软组织过度生长。多由垂体腺瘤引起。成人后GH过多表现为肢端肥大症。 |
| 急性黄色肝萎缩 | 急性重型肝炎时肝细胞大片坏死，肝体积急剧缩小，质软，切面黄或红黄。临床可出现急性肝衰竭。 |
| 动脉瘤 aneurysm | 动脉壁局限性、持久性异常扩张。常因动脉壁中膜薄弱或破坏。可由动脉粥样硬化、梅毒、先天缺陷等引起。 |

## 简答题专项

### 1. 肿瘤细胞异型性

答题模板：
1. 肿瘤细胞大小、形态不一，排列紊乱，极性消失。
2. 核异型性明显：核增大、深染，核浆比增大，核形不规则，核仁明显。
3. 核分裂像增多，可见病理性核分裂像。
4. 分化越差，异型性越大，恶性程度越高。

### 2. 肿瘤异型性与分化的关系

| 项目 | 分化 | 异型性 |
|---|---|---|
| 含义 | 肿瘤细胞接近来源正常细胞的程度 | 肿瘤与正常组织在结构/细胞形态上的差异 |
| 与恶性程度 | 分化越低，恶性越高 | 异型性越大，恶性越高 |
| 关系 | 反向相关 | 反向相关 |
| 举例 | 高分化鳞癌可见角化珠 | 低分化癌核深染、核分裂多、极性消失 |

### 3. 良性肿瘤与恶性肿瘤区别

| 项目 | 良性 | 恶性 |
|---|---|---|
| 分化 | 好 | 差 |
| 异型性 | 小 | 大 |
| 生长 | 慢，膨胀性或外生性 | 快，浸润性 |
| 边界/包膜 | 清楚，常有包膜 | 不清，常无包膜 |
| 继发改变 | 少 | 常出血、坏死、溃疡 |
| 转移 | 无 | 可有 |
| 复发 | 少 | 多 |
| 对机体影响 | 压迫、梗阻、激素作用 | 破坏、转移、恶病质、死亡 |

### 4. 癌和肉瘤区别

| 项目 | 癌 | 肉瘤 |
|---|---|---|
| 来源 | 上皮组织 | 间叶组织 |
| 发病率 | 多见 | 较少 |
| 年龄 | 中老年多 | 青少年相对多 |
| 肉眼 | 灰白、质硬、干燥 | 灰红、鱼肉状、质软 |
| 镜下 | 癌巢，实质与间质分界清 | 弥漫排列，血管丰富 |
| 转移 | 淋巴道为主 | 血道为主 |
| 免疫标志 | CK阳性 | Vimentin阳性 |

### 5. 肿瘤扩散方式

1. 直接蔓延/局部浸润：癌细胞侵入周围组织，如胃癌侵及浆膜、胰腺。
2. 淋巴道转移：癌常见，如乳腺癌转移至腋窝淋巴结，胃癌可转移至Virchow淋巴结。
3. 血道转移：肉瘤常见，癌也可见；肝、肺常为血道转移靶器官。
4. 种植性转移：体腔内播散，如胃癌种植腹膜和卵巢形成Krukenberg瘤。

### 6. 炎症基本病理变化

| 基本病变 | 含义 | 例子 |
|---|---|---|
| 变质 | 炎症局部组织变性和坏死 | 病毒性肝炎肝细胞坏死 |
| 渗出 | 液体、蛋白和炎细胞进入组织/体腔/表面 | 大叶性肺炎纤维素渗出 |
| 增生 | 实质和间质细胞增生 | 炎性息肉、肉芽肿、慢性炎症纤维组织增生 |

### 7. 脓肿和蜂窝织炎

| 项目 | 脓肿 | 蜂窝织炎 |
|---|---|---|
| 性质 | 局限性化脓性炎 | 弥漫性化脓性炎 |
| 组织破坏 | 明显，形成脓腔 | 组织间隙弥漫浸润，边界不清 |
| 常见菌 | 金黄色葡萄球菌 | 溶血性链球菌 |
| 常见部位 | 皮下、肝、脑、肺等 | 疏松结缔组织、皮下、肌间、阑尾 |
| 题眼 | 脓腔、局限、液化坏死 | 弥漫、扩散、透明质酸酶/链激酶 |

### 8. 坏疽类型和形态特点

答题顺序：先定义，再分三型。

1. 干性坏疽：四肢末端多见，动脉阻塞而静脉回流尚可，水分少，病灶干缩黑褐，分界清楚，中毒轻。
2. 湿性坏疽：肠、肺、子宫、胆囊多见，动静脉均受阻，病灶湿润肿胀、恶臭、分界不清，中毒重。
3. 气性坏疽：深部肌肉创伤合并产气厌氧菌感染，组织蜂窝状、捻发音，中毒严重。

### 9. 血栓形成条件、类型和结局

**条件**：内皮损伤、血流状态改变、血液高凝。

**类型**：
- 白色血栓：血流快处，血小板为主。
- 混合血栓：红白相间层状，静脉血栓体部、附壁血栓。
- 红色血栓：静脉血栓尾部，红细胞多。
- 透明血栓：微循环，纤维蛋白为主，DIC。

**结局**：软化溶解吸收、机化、再通、钙化、脱落栓塞。

### 10. 肺动脉血栓栓塞 vs 体循环动脉血栓栓塞

| 项目 | 肺动脉血栓栓塞 | 体循环动脉血栓栓塞 |
|---|---|---|
| 栓子来源 | 下肢深静脉、盆腔静脉、右心 | 左心附壁血栓、动脉粥样硬化斑块、动脉瘤 |
| 栓塞部位 | 肺动脉主干/分支 | 脑、肾、脾、下肢、肠系膜动脉等 |
| 后果 | 小栓子可无症状；大栓子可猝死；肺淤血时可出血性梗死 | 相应器官梗死，如脑梗、肾梗、脾梗、下肢坏疽 |
| 共同点 | 均为血栓脱落随血流阻塞血管 | 均可导致缺血、梗死或猝死 |

### 11. 贫血性梗死和出血性梗死

| 项目 | 贫血性梗死 | 出血性梗死 |
|---|---|---|
| 好发 | 心、肾、脾 | 肺、肠 |
| 条件 | 组织致密、侧支少、终末动脉供血 | 双重血供/侧支丰富、组织疏松、严重淤血 |
| 肉眼 | 灰白、楔形、边界较清 | 暗红、出血明显 |
| 镜下 | 凝固性坏死为主，脑例外为液化性坏死 | 坏死组织内大量红细胞 |

### 12. 大叶性肺炎、小叶性肺炎、间质性肺炎

| 项目 | 大叶性肺炎 | 小叶性肺炎 | 间质性/病毒性肺炎 |
|---|---|---|---|
| 性质 | 纤维素性炎 | 急性化脓性炎 | 间质性炎 |
| 范围 | 一个肺叶或大部 | 以细支气管为中心的散在小灶 | 肺泡间隔为主 |
| 渗出 | 肺泡内纤维素和中性粒细胞 | 细支气管、肺泡内中性粒细胞 | 淋巴细胞、单核细胞浸润 |
| 阶段 | 充血水肿、红肝样、灰肝样、溶解消散 | 可融合，常见于老人小儿体弱者 | 包涵体是确诊病毒性肺炎重要依据 |
| 并发症 | 肺肉质变、胸膜炎、肺脓肿、败血症 | 肺脓肿、脓胸、支扩、呼吸/心衰 | 透明膜、继发感染 |

### 13. 门脉性、坏死后性、胆汁性肝硬化

| 项目 | 门脉性肝硬化 | 坏死后性肝硬化 | 胆汁性肝硬化 |
|---|---|---|---|
| 常见病因 | 病毒性肝炎、慢性酒精中毒等 | 大片肝坏死后 | 胆道阻塞或自身免疫胆管损伤 |
| 肉眼 | 小结节为主，大小较一致 | 大小不等结节，纤维隔宽 | 肝绿色或黄绿色，胆汁淤积 |
| 镜下 | 纤细纤维隔分割，假小叶较均匀 | 宽大纤维隔，肝细胞再生结节大小不一 | 胆管增生、胆汁淤积、胆栓 |
| 临床 | 门脉高压、肝功能减退 | 肝功能损害重 | 黄疸明显，皮肤瘙痒 |

### 14. 结核基本病变与转化规律

**基本病变**：
1. 渗出：浆液纤维素性渗出，早期或免疫低下时明显。
2. 增生：上皮样细胞、Langhans巨细胞、淋巴细胞构成结核结节。
3. 坏死：干酪样坏死是结核特征性改变。

**转化**：
- 机体免疫强、菌量少：增生为主，纤维化/钙化，趋于愈合。
- 菌量多、毒力强、免疫低：渗出坏死为主，可液化、播散、空洞形成。

**原发性 vs 继发性肺结核**

| 项目 | 原发性 | 继发性 |
|---|---|---|
| 人群 | 儿童、初次感染 | 成人、再次感染或内源复燃 |
| 部位 | 肺上叶下部或下叶上部近胸膜 | 肺尖或锁骨下 |
| 特点 | 原发综合征：原发灶 + 淋巴管炎 + 肺门淋巴结结核 | 病灶局限，空洞常见，支气管播散 |
| 免疫 | 形成变态反应和免疫力 | 免疫基础上局部病变重 |

### 15. 肠结核、伤寒、菌痢、阿米巴病鉴别

| 疾病 | 好发部位 | 溃疡特点 | 题眼 |
|---|---|---|---|
| 肠结核 | 回盲部 | 环形溃疡，长轴与肠管长轴垂直 | 可致肠腔狭窄；结核肉芽肿、干酪样坏死 |
| 肠伤寒 | 回肠末端集合淋巴小结 | 椭圆形溃疡，长轴与肠管长轴平行 | 伤寒肉芽肿，髓样肿胀、坏死、溃疡、愈合四期 |
| 细菌性痢疾 | 乙状结肠、直肠 | 表浅不规则地图状溃疡 | 假膜性炎/纤维素性炎，黏液脓血便 |

### 18. 甲状腺乳头状癌

答题要点：
1. 大体多为质硬灰白结节，可伴囊性变或钙化。
2. 镜下见分支乳头状结构，乳头有纤维血管轴心。
3. 细胞核呈毛玻璃样/孤儿安妮眼核，可见核沟、核内假包涵体。
4. 间质常有砂粒体。
5. 易经淋巴道转移至颈部淋巴结，但预后较好。

### 19. 经典型霍奇金淋巴瘤

**基本特点**：淋巴结结构破坏，少量肿瘤性R-S细胞散在于丰富反应性背景中。背景可有小淋巴细胞、浆细胞、嗜酸性粒细胞、组织细胞等。

**经典型四型**：
1. 结节硬化型：胶原纤维分隔成结节，可见陷窝细胞。
2. 混合细胞型：R-S细胞较多，背景细胞混杂。
3. 富于淋巴细胞型：小淋巴细胞丰富，R-S细胞少。
4. 淋巴细胞消减型：淋巴细胞少，R-S细胞多，预后差。

> 结节性淋巴细胞为主型不是经典型。
